About Discovery

This website is one project in a network of research being conducted by API Evangelist. This toolkit is meant to provide you with news, analysis, tools, services and other resources that will help you learn about the world of APIs.

This is just one toolkit in a series:

  • Design
  • Deployment
  • Management
  • Evangelism
  • Monetization
  • Discovery
  • Consumption

Each one is meant to provide a focused area to learn about APIs, based upon your goals. If you are interested in learning about top APIs across many industries I recommend:

  • The API Stack

API Evangelist also provides projects that are based upon emerging API trends, in several areas:

  • Aggregation
  • Backend as a Service
  • Realtime
  • Reciprocity

All API Evangelist research projects use the Hacker Storytelling format and are openly licensed projects that you can download and fork on Github.

Feel free to use any of this research for yourself, just please make sure and spread the word and link back to relevant project sites.

If you have any questions about this research, contact me at info@apievangelist.com.
